HOW TO USE
===============================================================================

1. SETUP DATABASE
   - Create MySQL database 'q400_study'
   - Run SQL schema (see INSTALLATION.md)
   - Update config/database.php with credentials

2. CONFIGURE APPLICATION
   - Edit config/app.php for app settings
   - Copy .env.example to .env if needed

3. SETUP WEB SERVER
   - Apache: Enable mod_rewrite, configure VirtualHost
   - Nginx: Configure server block
   - PHP built-in: php -S localhost:8000

4. ACCESS APPLICATION
   - Homepage: /q400-study/public/
   - Login: /q400-study/public/login
   - Admin: /q400-study/public/admin

5. CREATE MODELS (Next Step)
   - Extend app/Core/Model
   - Define $table and $fillable properties
   - Use in controllers

6. BUILD FUNCTIONALITY
   - Create database schemas
   - Build model queries
   - Implement controller methods
   - Create views
